0. Preparacion cluster

    - Crearse una cuenta en dockerhub https://hub.docker.com/
    - habilitar k8s en docker desktop
    - instalar ingress controller:
        kubectl apply -f https://raw.githubusercontent.com/kubernetes/ingress-nginx/main/deploy/static/provider/cloud/deploy.yaml
        kubectl get pods -n ingress-nginx
    - Configurar dominio local
        Linux/Mac:
        cat /etc/hosts
        sudo nano /etc/hosts
        Guardar los cambios: Ctrl + O, Enter, Ctrl + X

        Windows
        C:\Windows\System32\drivers\etc\hosts
        Guardar los cambios y reiniciar
    - Crear namespace
        kubectl create namespace sbms

    - Configurar Helm y repositorio Bitnami
        curl https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-3 | bash

        helm repo add bitnami https://charts.bitnami.com/bitnami
        helm repo update
   
    - Crear instancia de PostgreSQL usando Helm y Bitnami repo
        helm install postgres bitnami/postgresql --namespace sbms --set auth.username=sbms --set auth.password=sup3rs3cur3p4ss --set auth.database=sbms_db

    - Crear instancia de RabbitMQ usando Helm y Bitnami repo
        helm install rabbitmq bitnami/rabbitmq --namespace sbms --set auth.username=admin --set auth.password=r4bb17sup3rp4ss --set auth.erlangCookie=r4bb17sup3rc00k13

    - Crear instancia de Redis usando Helm y Bitnami repo
        helm install redis bitnami/redis --namespace sbms --set auth.password=r3d1sr0ckz

3. Comando para interactuar con aks

    ### Cmds Genericos

    kubectl create namespace lic 

    kubectl apply -f file.ymal

    kubectl get deployments
    kubectl get services

    kubectl delete deployment nombre-deployment -n lic
    kubectl delete service nombre-service -n lic

    kubectl get pods -n lic

    kubectl logs sbms-customer-service-69dbddb4c6-62fkg -n sbms
    kubectl logs -f sbms-customer-service-69dbddb4c6-62fkg -n sbms

    ### Cmd ver ingress

    kubectl get ingress -n lic

    kubectl describe ingress sbms-ingress -n lic

    kubectl logs -l app.kubernetes.io/name=ingress-nginx -n ingress-nginx
